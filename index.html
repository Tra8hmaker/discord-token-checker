HOW TO GET TOKEN

説明 Overview

この方法ではブックマークレット（ブックマークとして保存できる小さなプログラム）を使用します。通常のブックマークと違い、クリックするとページ上で機能を実行します。 This method uses a bookmarklet (a small program saved as a bookmark). Unlike regular bookmarks, it executes functions when clicked.

パソコン版、スマホ版共通して使えるやり方です。この例ではChromeブラウザ（推奨）を使用したスマホ版を説明していきます。 This method works for both PC and mobile. This guide uses Chrome browser (recommended) on mobile as an example.

セキュリティについて： Security Notice: このやり方で得られるトークンという情報は、誰かと共有するとその人があなたのアカウントにログインできてしまいます。絶対に他人と共有しないでください。 If you share the token obtained through this method with someone, that person can log into your account. Never share it with others.

1. ブックマークレットをコピー 1. Copy Bookmarklet Code

2. 適当なページを準備としてブックマーク 2. Bookmark Any Page as Preparation

適当なページをブックマークします。後で編集してコードに置き換えます。 Bookmark any page. You'll edit it later and replace with the code.

3. ブックマークを編集 3. Edit the Bookmark

名前フィールドを消して、任意の名前に変更します（例：「トークン確認用」）。 Clear the name field and change it to any name you like (e.g., "Token Checker").

URLフィールドを削除し、Step 1でコピーしたコードを貼り付けて保存します。 Delete the URL field and paste the code from Step 1, then save.

4. Discordにログイン 4. Login to Discord

ブラウザ版でログインします。スマホアプリ、デスクトップアプリでは動作しません。必ずブラウザ版を使用してください。 Login in your browser. This won't work with mobile or desktop apps. You must use the browser version.

5. トークンを取得 5. Get Your Token

ログインした状態で、ブラウザの検索バーをタップします。 While logged in, tap the browser's address bar.

Step 3で設定したブックマーク名（例：「トークン確認用」）を検索し、ブックマークが見つかったらタップして実行します。 Search for the bookmark name you set in Step 3 (e.g., "Token Checker"), and once found, tap it to execute.

ポップアップが表示され、あなたのトークンが表示されます。 A popup will appear showing your token.
